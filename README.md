## 📋 Overview

This repository contains all data, code, and analysis for my MSc thesis research on machine learning studies on petrochronological experiment data from kula volcanic fields, Turkiye. The project integrates **EPMA** and **LA-ICP-MS** data to investigate How effective are the ensemble machine learning models for predicting the age of volcanic rocks from kula volcanic fields.
